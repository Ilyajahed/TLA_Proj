
Project Setup Instructions (Completed on 2025-05-25)

This file documents the exact steps followed to initialize and prepare the TLA_Proj project for LL(1) parser implementation using Python and GitHub.

---

1. ✅ Initialized a Local Git Repository
   Command:
       git init

2. ✅ Created Project Directory Structure
   Commands:
       mkdir grammar first_follow table dpda ast tests examples

3. ✅ Created and Configured Git Ignore
   File: .gitignore
   Contents:
       venv/
       __pycache__/
       *.py[cod]
       build/
       dist/
       *.egg-info/
       htmlcov/
       .coverage
       .cache
       *.dot

4. ✅ Created `run_tests.sh` Script
   File: run_tests.sh
   Contents:
       #!/usr/bin/env bash
       echo "Running all tests..."
       pytest -q
   Command to make executable:
       chmod +x run_tests.sh

5. ✅ Created `requirements.txt` (initially empty)

6. ✅ Initialized Python Virtual Environment
   Command:
       py -m venv venv

7. ✅ Activated Virtual Environment
   Command (Windows CMD):
       venv\Scripts\activate.bat

8. ✅ Installed Required Python Packages
   Command:
       pip install pytest graphviz

9. ✅ Saved Dependencies to `requirements.txt`
   Command:
       pip freeze > requirements.txt

10. ✅ Created `__init__.py` in All Python Package Folders
    Commands (Windows CMD):
        type nul > grammar\__init__.py
        type nul > first_follow\__init__.py
        type nul > table\__init__.py
        type nul > dpda\__init__.py
        type nul > ast\__init__.py

11. ✅ Test Run to Confirm Setup Works
    Command:
        bash run_tests.sh
    Output:
        Running all tests...
        .                                                                        [100%]
        1 passed in 0.04s

12. ✅ Created Remote Repository on GitHub (Private)

13. ✅ Linked Local Repo to GitHub
    Commands:
        git remote add origin https://github.com/Ilyajahed/TLA_Proj.git
        git add .
        git commit -m "Initial commit - setup project structure"
        git branch -M main
        git push -u origin main
